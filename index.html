Apa itu FFMC & FWI?

FFMC

Fine Fuel Moisture Code

Mengukur tingkat kekeringan bahan bakar ringan di permukaan tanah — dedaunan gugur, ranting kecil, alang-alang. Bahan setipis ini bereaksi cepat terhadap panas dan angin, jadi nilainya bisa berubah drastis dalam hitungan jam.

Skala 0–101. Makin tinggi, permukaan makin kering dan makin gampang tersulut walau hanya dari percikan kecil.
FWI

Fire Weather Index

Menggabungkan kekeringan bahan bakar dengan kondisi cuaca (angin, kelembapan) untuk memperkirakan seberapa besar dan cepat api bisa membesar apabila kebakaran benar-benar terjadi. Ini indeks yang dipakai sebagai status siaga utama di dashboard ini.

Makin tinggi, makin besar potensi intensitas api dan makin sulit dikendalikan bila kebakaran terjadi.
